Bindings.js is a simple javascript html binding library.
its purpose is to provide a simple and easy “Modal View” into javascript.

NOTE:

this library is in ealy stages of development, it dose contain bugs.
this library uses javascript Object.observe, Object.__defineGetter__ and Object.__defineSetter__
use object-observe polyfill for browsers that dont suport Object.observe
